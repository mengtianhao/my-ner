## 4.数据集

### 4.1 zh_msra

**MSRA：**(Levow，2006)是一个中文数据集，并作为基准数据集。MSRA中的数据是从新闻领域收集的，并作为SIGNAN  Backoff  2006的共享任务，有三种类型的命名实体；

### 4.2 zh_ontoontes4

**OntoNotes4.0：**(Pradhan等人，2011)是一个中文数据集，由来自新闻领域的文本组成。OntoNotes4.0注释了18个命名实体类型。在本文中，我们采用与Wu等人(2019年)相同的数据分割；
